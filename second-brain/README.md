## 💡 Developer Guidelines
*   **Keep notes atomic**: One note per concept or task attempt.
*   **Link notes together**: Use `[[wikilinks]]` to construct an associative web of knowledge.
*   **Document attempts**: Keep logs of errors, command outputs, and what worked or failed. It will prevent repeat debugging.
